# My dotfiles for EndeavourOs with BSPWM

## To configure the dotfiles
    1 - Install stow 
    2 - Create a ~/dotfiles
    3 - Copy the files that you want on the dotfiles
    4 - Run stow --adopt .

## This dotfiles uses
    - Polybar
    - Rofi
    - Dunst
    - BSPWM
    - SXHKD
    - ZSH
    - picom
    - alacritty
